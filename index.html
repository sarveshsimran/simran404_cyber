Remote Input Injection Vulnerability in Air Keyboard iOS App Still Unpatched

📅 July 17, 2025 | 🧑‍💻 By Sarvesh Bharti

A critical vulnerability has been discovered in the Air Keyboard iOS app that allows attackers to remotely inject arbitrary input via crafted network packets. The issue stems from insecure parsing of incoming data over Wi-Fi communication.

Security researchers have warned that this bug may allow remote control of user devices, potentially leading to serious privacy and data compromise — especially on jailbroken devices. Apple has yet to respond regarding the flaw.

Until patched, users are strongly advised to uninstall or disable the app and block it via firewall or network controls.
